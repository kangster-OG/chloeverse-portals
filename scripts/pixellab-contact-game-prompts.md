# Contact Game PixelLab Prompt Pack

Use these prompts in PixelLab for the desktop `/contact` planet-dodge game.
After downloading a fresh sprite set, run `python3 scripts/normalize-contact-game-assets.py` to strip PixelLab's opaque speckle fills and re-center the subject on the 128x128 canvas.

## PixelLab settings

- tool: `Create M-XL image`
- direction: `None`
- transparent background: `on`
- width: `128`
- height: `128`
- export format: `PNG`

## Global style rules

- high-detail premium pixel art
- dense clean shading
- clean silhouette
- low-noise palette
- `24-color look`
- `no grain`
- `no speckle field`
- centered single subject
- no stars
- no scene
