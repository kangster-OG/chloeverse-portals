Chloeverse Main Landing - Backdrop Rainbow Tuning v2.2

What it does
- Reduces hot pink dominance in the background reveal palette (weighted color selection).
- Removes straight seams/angles by switching the opaque base to full-screen radial fields (no repeat),
  and keeping only soft repeating micro/mid blobs on top.
- Keeps everything curved (radial gradients only), and preserves existing cursor/mask behavior.

Apply
1) Extract this folder into your repo root.
2) Run apply_patch.cmd
3) Start dev: npm run dev -- --turbo
